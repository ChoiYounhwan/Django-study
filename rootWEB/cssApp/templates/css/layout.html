    position 속성
    - 텍스트, 이미지, 테이블 등의 요소를 문서에 배치할 때 사용하는 속성
    - 배치 : 정적(position:static;), 상대(position:relative;), 절대(position:absolute;), 고정위치(position:fix;)

    float 속성
    - 화면을 구성하는 요소 간의 관계를 고려하여 각 요소를 배치하는 방법
      : inherit(요소를 감싸는 부모 요소의 float 속성을 상속), left, right, none(요소를 떠 있지 않게하는 것)

플롯팅 속성

float 속성은 특정 요소를 떠 있게 만들어 줍니다. 여기서 '떠 있다'라는 말의 의미는 특정 요소가 기본 레이아웃에서 벗어나 문서의 왼쪽 또는 오른쪽에 이동하는 것 입니다. float 속성은 복잡한 형태의 레이아웃을 구성하는데 필요한 핵심 소성이며, 특정 요소가 주변 요소와 자연스럽게 어울리게 해 줍니다.
